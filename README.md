# next-boilerplate

A Next.js boilerplate with:

- TypeScript
- Jest
- Material Icons
- ESLint
- Prettier
- Tailwind CSS
- Storybook

## Getting Started

### Setup

1. Clone this repository.
2. Run `yarn install`.

### Usage

- `yarn dev`
- `yarn lint`
- `yarn test`
- `yarn storybook`
